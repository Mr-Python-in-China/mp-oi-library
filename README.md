# mp-oi-library

这个 vscode 插件提供了一些在算法竞赛中常见的算法库，并提供命令用于快速将内容填充进当前文件，以节省编写板子的时间。

## features

- [ST 表](./library/mrpython/sparse_table.hpp)
- [无标记线段树](./library/mrpython/typical_segment_tree.hpp)
- [懒标记线段树](./library/mrpython/lazy_segment_tree.hpp)
